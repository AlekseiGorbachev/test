## Операции git
- git pull - обновляет вашу локальную ветку из удаленного репозитория. Это комбинация двух команд git fetch (забирает изменения из удаленного репозитория) и git merge (объединяет изменения с вашей веткой) 
- git push
- git squash 
- git reset
